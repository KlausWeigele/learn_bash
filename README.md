## Struktur pro Thema

```
thema/
├── 00-EXPLORE.md    # Neugier wecken, Fragen stellen
├── CHEATSHEET.md    # Kompakte Referenz
└── scripts/         # Praktische Beispiele
```

## Wie ich lerne

1. **EXPLORE.md lesen** - Fragen stellen, nachdenken
2. **Skripte ausprobieren** - `chmod +x script.sh && ./script.sh`
3. **Cheatsheet nutzen** - beim Üben nachschlagen
4. **Eigene Skripte schreiben** - das Gelernte anwenden

## Umgebung

- macOS mit Zsh als Standard-Shell
- Terminal.app oder iTerm2
- VS Code für Skript-Bearbeitung
